图标素材网站

Katorly2021-05-06

记录几个能在线获取且可免费获得的图标素材网站，用于建站、设计等。
排名不分先后，仅依照记录的时间顺序排序。本文中各网站的协议和其它限制可能会有变更，仅供参考，请以各个网站的实际规定为准。不定期更新。

完全免费的

Bootstrap Icons

⚠标出处 ✔商用

Bootstrap官方图标库。

格式: .svg
地址：icons.getbootstrap.com
协议：MIT

Google Material Icons

⚠标出处 ✔商用

可在线变换大小和颜色。

格式: .svg
地址：material.io/resources/icons
协议：Apache license 2.0

Icofont

⚠标出处 ✔商用

选中图标后选择“Download SVG”即可下载。

格式: .svg
地址：icofont.com
协议：CC BY 4.0

Feather

⚠标出处 ✔商用

可在线变换大小和颜色，不过图标数量较少。

格式: .svg
地址：feathericons.com
协议：MIT

Ionicons

⚠标出处 ✔商用

选中图标后选择“SVG”即可下载。（若浏览器窗口过小可能不会显示该按钮）

格式: .svg
地址：ionic.io/ionicons
协议：MIT

CSS.gg

⚠标出处 ✔商用

选定图标后在右边紫色按钮显示“.xd”的地方按下，在其弹出的下拉菜单中可选择格式，选好后点击“Download”即可下载。

格式: .css,.svg,.tsx,.fig,.xd,.json,.xml
地址：css.gg
协议：MIT

字节跳动 IconPark

⚠标出处 ✔商用

还支持React和Vue图标代码组件。

格式: .png,.svg
地址：iconpark.oceanengine.com
协议：Apache License 2.0

Octicons

⚠标出处 ✔商用

Github官方图标库。
图标较少，因为其只提供Github网站上使用的图标。

格式:.svg
地址：primer.style/octicons
协议：MIT

useAnimations

⚠标出处 ✔商用

提供动态（自动播放或交互式）图标。

格式: .svg,.json
地址：useanimations.com
协议：CC BY 4.0

iconmonstr

👍不需标出处 ✔商用

值得注意的是，该网站特别反对AdBlocker，在访问网站时不停地弹出警告。

格式: .svg,.png
地址: iconmonstr.com
协议: Iconmonstr协议

Fontello

??出处 ??商用

作者还在Wiki首页给出了一些CC协议许可的其它图标库。

格式: 字体
地址: fontello.com
协议: 混合，使用前请先阅读每种图标包的协议

部分收费的

FontAwesome

⚠标出处 ✔商用

最流行的图标库之一，被广泛使用。

下载免费图标的方法（不需要导入它的库，免费版限制太多）：选择一个免费图标，右上角点击“Copy SVG code”或“Download SVG file”。

格式: .svg
地址：fontawesome.com/icons
协议：CC BY 4.0

Icons8

🚨限制多 ⚠标出处 ❌商用

图标较多，且在不断更新，支持在线编辑。
.svg格式仅他人商标的图标免费。

（免费下载其.svg文件的方法已失效）
绕过其下载限制的免费方法：先在电脑浏览器最大化窗口中选好图标，并打开图标的对应页面（只显示该图标的页面），然后还原窗口（把窗口缩小），右键图片即可保存。
免费下载其svg的方法：先打开图标的对应页面（只显示该图标的页面），点击“Recolor”（重新着色），确认图没有发生变化后，点击“Done”（完成），右键图标即可下载svg。

格式: .png
地址：icons8.com/icons
协议：免费使用，但要加入ICONS8链接

Flaticon

⚠标出处 ✔商用

付费格式包含.svg。

格式: .png
地址：flaticon.com
协议：免费使用，但要加入FLATICON链接

IcoMoon

??出处 ??商用

可自由组合各种图标，打包下载或转为css。
部分图标包收费。

格式: .svg,.png,字体
地址：icomoon.io
协议：混合，使用前请先阅读每种图标包的协议

Lordicon

🚨限制多 ⚠标出处 ✔商用

提供大量动态图标，支持在线编辑。

格式: .json,.gif
地址：lordicon.com
协议：免费版 CC BY ND 4.0

Noun Project

⚠标出处 ✔商用

仅图标可商用，照片免费版不可商用。
在线编辑收费。

格式: .svg,.png
地址: thenounproject.com
协议: CC BY 3.0

Iconfinder

⚠标出处 ✔商用

图标较多，免费图标也多。

格式: .svg,.png,.ico,.ai,.icns
地址: iconfinder.com
协议: CC BY 4.0

Iconshock

🚨限制多 ⚠标出处 ❌商用

图标样式齐全。

格式: .png
地址: iconshock.com
协议: 免费使用，但要加入Iconshock链接

Streamline

⚠标出处 ✔商用

支持在线编辑。

格式: .png
地址: streamlinehq.com/icons
协议: 免费使用，但要加入Streamline链接



（最后，感谢 @dalia09sultana 建议了部分图标素材网站）
